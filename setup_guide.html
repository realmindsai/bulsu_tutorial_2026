AI Tools Setup Guide

GDG x Bulacan State University AI Workshop | 20 February 2026

This guide gets you set up with AI coding tools. Takes about 15-30 minutes.

1. Package Managers

You need package managers to install and run code. Install both of these.

1a Python: Install uv

Windows (PowerShell):

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

Mac / Linux:

curl -LsSf https://astral.sh/uv/install.sh | sh

Restart your terminal, then verify:

uv --version

1b JavaScript: Install Node.js

Windows: Download and run the installer from nodejs.org

Mac:

brew install node

Linux:

curl https://get.volta.sh | bash
exec bash
volta install node

Verify:

node --version

2. AI Coding Tools

Install at least one of these. All are free with a subscription or free tier.

2a Gemini CLI (Recommended - Free)

Free with any Google account.

npm install -g @google/gemini-cli

Verify and log in:

gemini --version
gemini

When prompted, select "Login with Google". It will print a URL - copy it, paste into your browser, sign in with your Google account, approve, and come back to the terminal.

Alternative: Get a free API key from aistudio.google.com/apikey and set it:

export GEMINI_API_KEY="your-key-here"

2b Claude Code (Optional - Requires Subscription)

Requires a Claude Pro or Max subscription.

npm install -g @anthropic-ai/claude-code

Verify and log in:

claude --version
claude

Same browser login flow as Gemini - it prints a URL, you copy it to your browser, sign in, approve, come back.

2c OpenAI Codex (Optional - Requires Subscription)

Requires a ChatGPT Plus, Pro, Team, or Edu subscription.

npm install -g @openai/codex

Verify:

codex --version

3. The Login Pattern

All three tools follow the same flow:

Run the tool → It prints a URL → Copy the URL → Paste into browser → Sign in and approve → Get a code → Paste code back in terminal → Done

You only do this once per tool. After that, it remembers you.

4. Verification Checklist

Run each command and confirm you see a version number:

  • uv --version
  • node --version
  • gemini --version
  • claude --version (only if installed)
  • codex --version (only if installed)

If any command says "command not found", try restarting your terminal first.

5. Troubleshooting

"Command not found" after installing

The installer added the tool to your PATH, but your terminal does not know yet.

Fix: Close your terminal and open a new one. On Mac/Linux you can also run exec bash.

npm install fails with permission errors

Windows: Run PowerShell as Administrator (right-click → Run as administrator).

Mac/Linux: Do NOT use sudo npm install -g. Instead, fix npm permissions:

mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
exec bash

Need to update a tool later

npm update -g @google/gemini-cli
npm update -g @anthropic-ai/claude-code
uv self update
