Setup

You have two tools to choose from. Both work well. Pick whichever one feels right, or try both.

Step 1: Install OpenCode

curl -fsSL https://opencode.ai/install | bash

Or if you have npm:

npm install -g opencode-ai

Step 2: Set your API key

You received a unique API key (it looks like sk-AbCdEf123456...). Set it as an environment variable:

Mac/Linux:

export LITELLM_API_KEY=sk-YOUR-KEY-HERE

To make this permanent, add that line to your ~/.bashrc or ~/.zshrc file.

Windows (PowerShell):

$env:LITELLM_API_KEY = "sk-YOUR-KEY-HERE"

Step 4: Launch OpenCode

cd your-project-folder
opencode

Done. You're now talking to an AI that can read, write, and edit your code.

Switching models

Inside OpenCode, type /model and select a different model:

/model litellm/openrouter/deepseek/deepseek-v3.2     # cheap everyday coding
/model litellm/openrouter/qwen/qwen3-coder            # code specialist
/model litellm/openrouter/anthropic/claude-sonnet-4.5  # premium quality
/model litellm/openrouter/deepseek/deepseek-r1         # step-by-step reasoning

Step 1: Install Aider

pip install aider-chat

Step 2: Create a configuration file

Download .aider.conf.yml

Save this file to your home directory and edit it to add your API key. Or create it manually:

Mac/Linux: ~/.aider.conf.yml
Windows: C:\Users\YourName\.aider.conf.yml

openai-api-base: http://170.64.164.75:4000/v1
openai-api-key: sk-YOUR-KEY-HERE
model: openai/openrouter/deepseek/deepseek-v3.2

Replace sk-YOUR-KEY-HERE with your actual API key.

Step 3: Launch Aider

cd your-project-folder
aider

Switching models

Inside an aider session, use the /model command:

/model openai/openrouter/deepseek/deepseek-v3.2        # cheap everyday coding
/model openai/openrouter/qwen/qwen3-coder               # code specialist
/model openai/openrouter/anthropic/claude-sonnet-4.5     # premium quality
/model openai/openrouter/deepseek/deepseek-r1            # reasoning mode

Or launch with a specific model:

aider --model openai/openrouter/anthropic/claude-sonnet-4.5

Try these now

Once you have OpenCode or Aider running, try these:

1. Hello World

Make me a Python script that asks for my name and greets me in both Filipino and English. Make it colorful in the terminal.

2. Fix this bug

Create a file called buggy.py with this code, then ask the AI to find and fix the bug:

def average(numbers):
    total = 0
    for n in numbers:
        total += n
    return total / len(numbers)

print(average([]))
There is a bug in buggy.py. Find it and fix it.

3. Explain mystery code

Save this to a file and ask:

f=lambda n:n if n<2 else f(n-1)+f(n-2)
print([f(i)for i in range(15)])
Explain this code to me line by line. What does it do? How does it work?

4. Build a tool

Create a simple command-line calculator in Python that can handle +, -, *, / and remembers the last result so I can chain calculations.

5. Go wild

Ask it to build something you've always wanted but didn't know how to make. A game, a web scraper, a to-do app. The $20 goes further than you'd expect.

Check your remaining budget

To see how much credit you have left, run this in your terminal:

curl -s http://170.64.164.75:4000/key/info \
  -H "Authorization: Bearer sk-YOUR-KEY-HERE" | python3 -m json.tool

Look for "spend" (how much you've used) and "max_budget" (your total: 20.0).

Remaining = max_budget - spend
